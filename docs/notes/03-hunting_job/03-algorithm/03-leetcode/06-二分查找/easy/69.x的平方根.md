## 69. x 的平方根

[力扣原题链接（点我直达）](https://leetcode-cn.com/problems/sqrtx/)

实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

### 相乘穷举，可能溢出

```C++
    int mySqrt(int x) {
	if (x == 0) return 0;
	if (x <= 3) return 1;
	int i = 2;//输入2147395600时报错，（i*i）也是个int，				//int，可保存不了这么大的数字，只好改为					//long
	for (; i < x / 2; ++i)
	{
		if ( (i * i <= x) && ((i + 1)* (i + 1) > x))
			return i;
	}
    return i;
    }
```

### 二分法会更快一点

```c++
    int mySqrt(int x) {
	if (x == 0) return 0;
	if (x <= 3) return 1;
	int min = 0;
	int max = x;
	while (max - min > 1)
	{
		int m = (max + min) / 2;
		if (x / m < m)//用x/m<m，而不是m*m<x可以防止溢出
			max = m;
		else
			min = m;
	}
	return min;
    }
```





```C++
    int mySqrt(int x) {
        long mid = 0;
        long left = 0;
        long right = x;
        while (left < right)
        {
            mid = (left + right + 1) / 2;
            long sq = mid * mid; //或者这样也会更快一点
            if (sq > x)
            {
                right = mid - 1;
            }
            else
            {
                left = mid;
            }
        }
        
        return (int)left;
    }
```
